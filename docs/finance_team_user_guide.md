# Financial Export Process: User Guide for Finance Team

This guide provides instructions for the finance team on how to initiate, monitor, and troubleshoot the optimized financial export process.

## 1. Initiating the Export

*   **Step 1:** Navigate to the export portal via [Link to Export Portal].
*   **Step 2:** Select the 'Quarterly Financial Audit Export' option.
*   **Step 3:** Choose the desired fiscal quarter from the dropdown menu.
*   **Step 4:** Click 'Initiate Export'.

## 2. Monitoring Export Progress

*   After initiating, you will be redirected to the 'Export Status' page.
*   The status will update automatically. You can refresh the page manually if needed.
*   Expected statuses include: 'Pending', 'Processing', 'Complete', 'Failed'.

## 3. Troubleshooting Common Issues

*   **Export Timed Out:** This issue should be resolved with the new process. If it occurs, please notify the IT team immediately with the export ID.
*   **Incomplete Data:** Verify the selected fiscal quarter. If correct, escalate to IT.
*   **Permissions Error:** Ensure you have the 'Financial Auditor' role. If you believe this is an error, contact your system administrator.

For any unlisted issues, please contact the IT support team at [IT Support Contact Information] and provide the export ID and a description of the problem.
