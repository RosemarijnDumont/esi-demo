# Financial Export Process - User Guide

## 1. Introduction
This guide provides instructions for the finance team on how to efficiently use the optimized financial export process. This new process addresses previous timeout issues and ensures a complete and accurate dataset on the first attempt.

## 2. Accessing the Export Tool

*   **System Name:** Financial Data Export System (FDES)
*   **URL:** `https://fdes.yourcompany.com`
*   **Login:** Use your standard corporate credentials.

## 3. Exporting Financial Data

1.  **Navigate to Export Section:** After logging in, click on the "Financial Exports" tab in the left-hand navigation menu.
2.  **Select Export Type:** Choose the desired export type from the dropdown menu:
    *   `Quarterly Financial Report`
    *   `General Ledger Details`
    *   `Custom Date Range Export`
3.  **Specify Parameters:**
    *   **Quarterly Report:** Select the desired fiscal quarter and year.
    *   **General Ledger:** Select the start and end dates, and optionally, specific accounts.
    *   **Custom Date Range:** Enter the precise start and end dates using the date picker.
4.  **Initiate Export:** Click the "Generate Export" button. The system will display a confirmation message and begin processing the request.
5.  **Monitor Export Status:** You can monitor the status of your export in the "Export History" section. The status will transition from "Processing" to "Completed" or "Failed".
6.  **Download Exported Data:** Once the status is "Completed", click the "Download" icon next to your export to retrieve the data in CSV format.

## 4. Troubleshooting Common Issues

*   **Export stuck in "Processing":**
    *   Wait a few minutes and refresh the page. Large exports may take longer.
    *   If it persists, contact IT support with the export ID.
*   **"Failed" Export Status:**
    *   Review the error message displayed in the Export History. It may provide a specific reason.
    *   Verify your selected parameters are correct (e.g., valid date ranges).
    *   If the issue continues, contact IT support with the export ID and error message.
*   **Incomplete Data:**
    *   Ensure all required parameters were specified correctly for the chosen export type.
    *   If you suspect data corruption or missing records, contact IT support.
